Pitch is honored only by local/system voices (e.g. macOS and Windows desktop voices). Many premium voices — Google, "Natural"/"Neural", and Siri — ignore pitch entirely, so this slider may have no audible effect when one of those is selected. Pick a ★ or ★★ system voice if you need the two speakers to differ in pitch.

Cloud TTS (not enabled in this version). This build uses your browser's built-in voices and costs nothing. A future revision can route synthesis to a cloud provider for more natural voices — the code has a single drop-in point marked CLOUD TTS DROP-IN POINT.
